# LFO in Kick Drum Synthesis

## Key Techniques

### 1. Pitch Envelope Decay
- Start high, drop low quickly (e.g., 150Hz → 40Hz)
- Creates punchy attack with sub body
- More effective than static pitch

### 2. LFO for Pitch Modulation
- Use LFO to modulate kick pitch
- Creates "wobble" or "pitch bend" effect
- Common in dubstep, trap, modern EDM

### 3. LFO Rate for Kicks
- **Slow (0.5-2 Hz):** Subtle movement
- **Medium (2-8 Hz):** Classic wobble
- **Fast (8-16 Hz):** More aggressive

### 4. LFO Shape for Kicks
- **Sine:** Smooth pitch bend
- **Saw/ Square:** More aggressive
- **Triangle:** Classic 808 style

## Tools
- **Ableton Operator** - Built-in kick synthesis
- **NI Massive** - LFO + envelope control
- **Serum** - Wavetable + LFO
