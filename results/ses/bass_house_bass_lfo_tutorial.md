# Bass House Bass with LFO - Tutorial

## Bass House Bass Characteristics

- **Groove-oriented** - Rhythmic, syncopated patterns
- **Mid-range forward** - Not just sub, texture matters
- **Distorted/Resonant** - Often has "grit"
- **Pumping** - Works with kick in a groove

## LFO Usage on Bass House Bass

### 1. Filter LFO (Most Common)
```
Rate: 1/8 to 1/4 (sync to tempo)
Depth: 30-60%
Shape: Saw or Square
```
- Creates "wah" effect
- Adds groove to bass line

### 2. Pitch LFO (Subtle)
- Rate: 0.5-2 Hz
- Depth: 2-5 cents
- Adds "movement" to sub

### 3. LFO for "Pump"
- Use on amplitude
- Sync to kick pattern
- Creates bass "breathing"

## Step-by-Step: Bass House Bass

### Step 1: Basic Sound
- Oscillator: Sawtooth (classic bass house)
- Filter: Lowpass at 200-400Hz
- Resonance: 20-30%

### Step 2: Add Texture
- Layer second oscillator (one octave up)
- Add slight distortion
- Slight detune for "chorus" effect

### Step 3: LFO Application
- **Filter LFO:** 1/8 rate, 40% depth
- **Pitch LFO:** 1Hz, 3 cents (subtle)
- **Amp LFO:** 1/4 rate for pumping

### Step 4: Sidechain
- Light sidechain to kick
- 4:1 ratio, fast attack, medium release

### Step 5: Finishing
- Compressor: 2:1, 10ms attack, 100ms release
- Clipper: subtle for "bite"
- EQ: boost 200-400Hz for presence

## LFO Settings Summary

| Effect | Rate | Depth | Shape |
|--------|------|-------|-------|
| Filter Movement | 1/8 - 1/4 | 30-60% | Saw |
| Sub Movement | 0.5-2 Hz | 2-5% | Sine |
| Pump/Groove | 1/4 - 1/2 | 50-100% | Square |
| Texture | 1/1 | 10-20% | Triangle |

## Common Mistakes

1. **Too much LFO** - Bass house is about groove, not wobble
2. **Clean bass only** - Needs some grit/distortion
3. **Ignoring mid-range** - Bass house needs presence (200-400Hz)
4. **No sidechain groove** - Kick-bass relationship is key

## Creating the "Pump" Effect

1. Create LFO on filter cutoff
2. Sync to kick (1/4 or 1/8)
3. Add envelope to LFO depth
4. Result: bass that "breathes" with kick
