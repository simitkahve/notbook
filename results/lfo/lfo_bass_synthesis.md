# LFO in Bass Synthesis

## Key Techniques

### 1. Sub-Bass with LFO
- LFO modulates filter cutoff for movement
- Add subtle pitch modulation for "alive" feel
- Common: 0.1-1 Hz rate, low amount

### 2. Reese Bass & Layering
- Reese bass = layered detuned saws
- LFO on filter creates "wah" effect
- Add LFO on pitch for wobble

### 3. FM Sub Bass
- Use FM for harmonic-rich sub
- LFO modulates FM index for movement
- Serum, Operator good for this

### 4. Pitch Envelope vs LFO
- **Envelope:** One-time pitch drop
- **LFO:** Continuous pitch wobble
- Combine both for character

### 5. LFO Recommendations for Bass
- **Rate:** 0.1-4 Hz
- **Shape:** Sine (smooth), Saw (aggressive)
- **Amount:** Low (2-5 cents) for sub, higher for synth bass
- **Sync to tempo:** Quarter note, half note

## Tools
- **Serum** - Wavetable + comprehensive LFO
- **Massive / Massive X** - FM synthesis + LFO
- **Analog synths** - Built-in LFO

## Best Practices
- Keep sub-bass clean (minimal LFO)
- Use high-pass filter on any processing
- Reference in mono
- Don't over-modulate
